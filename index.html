FEATURES

Courses

Create full featured educative courses. Access all essential features like student management, lesson management and customisation.

Media

Use images, videos and PDF files to explain complex topics in a lucid fashion. You can even embed YouTube videos.

Payments

Let students buy your courses. CourseLit supports Stripe out of the box for payment processing. More payment methods are coming soon.

Blogs

You don't need a separate system to publish tutorials and blog posts. Using CourseLit, you can create posts right from its dashboard.
